## About This Project

This is a **Next.js CMS template** that provides a simple, file-based content management system. It's designed to be a starting point for building content-driven websites.

### Features

- **File-based CMS** — manage content through Markdown and MDX files
- **Blog support** — built-in blog with listing and individual post pages
- **Static pages** — create standalone pages easily
- **Tailwind CSS** — utility-first styling out of the box
- **TypeScript** — full type safety throughout
- **SEO-friendly** — server-rendered pages with dynamic metadata

### Technology Stack

- [Next.js](https://nextjs.org) — React framework
- [Tailwind CSS](https://tailwindcss.com) — Styling
- [MDX](https://mdxjs.com) — Markdown with JSX support
- [gray-matter](https://github.com/jonschlinkert/gray-matter) — Frontmatter parsing
- [TypeScript](https://www.typescriptlang.org) — Type safety

### Extending the CMS

This template can be extended in several ways:

- **Add a headless CMS** — integrate with Sanity, Strapi, Contentful, or any other headless CMS
- **Add authentication** — protect admin routes with NextAuth.js or similar
- **Add a database** — store content in a database instead of files
- **Add search** — implement full-text search across your content
